Welcome to Your Personal Library

This application is mean to allow you to track books that you have read or used for school courses. Why would you need this, well I find I don't like to keep hard copies of books and I can never remember what I have read until I am about a chapter in. So instead of spending money on a book, only to find out I have read it, I developed this application. To use the application you will need to create an account. You can do so at the website at http://apps.mfwd.net/PersonalLibrary or you can use the register form here. The account is free and so is the use of the website. This will not change. I may eventually charge $0.99 for the app but for now it is free. I don't expect a lot of usage, which is why this is possible. I mainly built this app so I can learn how to build mobile applicaitons. Feel free to contact me and let me know if you have changes you would like. I may open source the project so others can help. For now it is just me, so it might be a while before I can change things.
